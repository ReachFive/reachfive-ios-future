== Description

Logs the user in using credentials stored in the keychain. 
The system displays all available credentials in a link:https://developer.apple.com/design/human-interface-guidelines/sheets[modal sheet^].
Credentials can be passwords and/or passkeys.

== Usage

Start this request in response to a user interaction when the display is `.Always` or automatically and/or early in a view lifecycle (e.g., in ``viewDidAppear``).
Alternatively, start the request during the application launch when the display is `.IfImmediatelyAvailableCredentials`.
